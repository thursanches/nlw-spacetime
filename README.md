<h1>About</h1>
An app (mobile and web) for reccording memorys. Is possible to add text, photos and videos to a timeline. 

<h1>To run:</h1>
<ol>
  <h2>On web</h2>
  This commands is for client and server.
  <li>

    npm install
  </li>
  <li>
    
    npm run dev
  </li>
  <h2>On mobile</h2>
  <li>

    npm install
  </li>
  <li>

    npm run start
  </li>
</ol>
